.NET FlameWork 4.8をインストールする必要があるので、以下を必ず実行してください！！

①以下の2つのファイルをMicrosoftのサイトからダウンロードしてください。

NDP48-x86-x64-AllOS-JPN.exe
NDP48-x86-x64-AllOS-JPN.exe

②ダウンロードしたら、「DotNetFX48」の名前でフォルダを作成してください。

③ダウンロードしたファイルを②のフォルダにコピーしてください。

④③のフォルダを“21 Filament Loading Device.msi”と“setup.exe”のファイルと同じフォルダに移動してください。
